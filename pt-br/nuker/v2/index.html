zufinho nuker V2

zufinho nuker V2, desenvolvido em fevereiro de 2024, desenvolvido com Python e a biblioteca requests.
Desenvolvido especialmente para windows (mas também uma versão para linux).

Na versão do windows ele vem com alguns arquivos python na pasta src, cada arquivo é para um determinada ação, como por exemplo um arquivo quando executado, cria canais, o outro cria cargos.
O programa principal executa esses programas de acordo com as configurações, sendo um pouco mais rapido que o v1, mas afetando mais no desempenho.
